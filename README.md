 Machine Learning Internship – Task 02

📖 Project Overview

This project is a part of my Machine Learning Internship. The objective of this task is to build a classification model that predicts whether a customer will purchase a product based on demographic and behavioral information. The project demonstrates the complete machine learning workflow, including data preprocessing, exploratory data analysis, model training, evaluation, and prediction.



🎯 Objective

Develop a supervised machine learning model to classify customer purchasing behavior using historical customer data and evaluate its performance using standard evaluation metrics.



📂 Dataset

The dataset includes customer information such as:

- Customer ID
- Gender
- Age
- Estimated Salary
- Purchased (Target Variable)

The target variable indicates whether the customer purchased the product.

📌 Project Workflow
